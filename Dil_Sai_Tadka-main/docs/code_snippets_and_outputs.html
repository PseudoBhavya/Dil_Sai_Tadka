1. Partner API — Authentication Filter

Every call to /api/partner/** is intercepted by PartnerAuthFilter.java. It validates the X-Partner-Key header, enforces rate limiting (60 req/min), and persists usage logs to PostgreSQL.

PartnerAuthFilter.java JAVA 
@Component
@RequiredArgsConstructor
@Slf4j
@Order(1)
public class PartnerAuthFilter extends OncePerRequestFilter {

    private final PartnerIntegrationRepository partnerIntegrationRepository;
    private final ApiUsageLogRepository apiUsageLogRepository;

    // In-memory rate limiter: partner + minute-bucket → count
    private final Map<String, AtomicLong> rateLimitCounter = new ConcurrentHashMap<>();
    private final long rateLimitMax = 60;

    @Override
    protected void doFilterInternal(HttpServletRequest request,
            HttpServletResponse response, FilterChain filterChain) {

        String apiKey = request.getHeader("X-Partner-Key");
        long startTime = System.currentTimeMillis();

        // ① Validate key presence
        if (apiKey == null || apiKey.trim().isEmpty()) {
            sendErrorResponse(response, 401, "Unauthorized: X-Partner-Key missing");
            return;
        }

        // ② Validate key against database
        PartnerIntegration partner = partnerIntegrationRepository
                .findByApiKey(apiKey).orElse(null);
        if (partner == null || partner.getStatus() != Status.ACTIVE) {
            sendErrorResponse(response, 401, "Unauthorized: Invalid or inactive API key");
            return;
        }

        // ③ Rate limiting (60 req/min per partner)
        long minuteKey = System.currentTimeMillis() / 60000;
        String rateKey = partner.getName() + ":" + minuteKey;
        rateLimitCounter.putIfAbsent(rateKey, new AtomicLong(0));
        long count = rateLimitCounter.get(rateKey).incrementAndGet();

        if (count > rateLimitMax) {
            sendErrorResponse(response, 429, "Too Many Requests: Rate limit exceeded");
            return;
        }

        // ④ Proceed and log to DB
        filterChain.doFilter(request, response);
        saveLogToDb(partner.getName(), endpoint, method, statusCode, startTime, null);
    }
}

            .build())
        .collect(Collectors.toList());
    return ResponseEntity.ok(list);
}

2.2 Creating — POST /api/partner/bookings

PartnerApiController.java — createBooking() JAVA 
@PostMapping("/bookings")
public ResponseEntity<?> createBooking(@RequestBody PartnerBookingDTO dto) {
    if (dto.getUserEmail() == null || dto.getRoomId() == null
            || dto.getCheckIn() == null || dto.getCheckOut() == null) {
        return ResponseEntity.badRequest()
                .body(Map.of("message", "userEmail, roomId, checkIn, checkOut required"));
    }
    User user = userRepository.findByEmail(dto.getUserEmail()).orElseGet(() ->
            userRepository.findByEmail("customer@dilsaitadka.com").orElseThrow());
    Room room = roomRepository.findById(dto.getRoomId()).orElseThrow();

    Booking booking = Booking.builder()
        .user(user).room(room)
        .checkIn(dto.getCheckIn()).checkOut(dto.getCheckOut())
        .guests(dto.getGuests() != null ? dto.getGuests() : 1)
        .totalAmount(dto.getTotalAmount() != null
                ? dto.getTotalAmount() : room.getPricePerNight() * 2)
        .status(Booking.BookingStatus.CONFIRMED)
        .build();

    Booking saved = bookingRepository.save(booking);
    return ResponseEntity.status(HttpStatus.CREATED).body(dto);
}

@PutMapping("/orders/{id}")
public ResponseEntity<?> updateOrder(@PathVariable Long id, @RequestBody PartnerOrderDTO dto) {
    var opt = orderRepository.findById(id);
    if (opt.isEmpty()) return ResponseEntity.status(404).body(Map.of("message", "Order not found"));
    Order o = opt.get();
    if (dto.getStatus() != null)
        o.setStatus(Order.OrderStatus.valueOf(dto.getStatus().toUpperCase()));
    if (dto.getTotalAmount() != null) o.setTotalAmount(dto.getTotalAmount());
    Order saved = orderRepository.save(o);
    return ResponseEntity.ok(dto);
}

3. Security Config — Role-Based Access Control

Spring Security is configured to restrict /api/bookings/** and /api/orders/** to ROLE_ADMIN, preventing customers from viewing each other's data.

SecurityConfig.java JAVA 
@Bean
public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
    http
        .cors(cors -> cors.configurationSource(corsConfigurationSource()))
        .csrf(csrf -> csrf.disable())
        .sessionManagement(sm -> sm.sessionCreationPolicy(STATELESS))
        .authorizeHttpRequests(auth -> auth
            .requestMatchers("/api/auth/**").permitAll()
            .requestMatchers("/api/partner/**").permitAll()  // ← protected by PartnerAuthFilter
            .requestMatchers(GET, "/api/menu/**").permitAll()
            .requestMatchers(GET, "/api/rooms/**").permitAll()
            // 🔒 Admin-only: protect all bookings & orders from regular users
            .requestMatchers(GET, "/api/bookings/**").hasRole("ADMIN")
            .requestMatchers(GET, "/api/orders/**").hasRole("ADMIN")
            .requestMatchers(PUT, "/api/bookings/**/status").hasRole("ADMIN")
            .requestMatchers(PUT, "/api/orders/**/status").hasRole("ADMIN")
            .requestMatchers("/api/dashboard/**").hasRole("ADMIN")
            .requestMatchers("/api/users/**").hasRole("ADMIN")
            .anyRequest().authenticated()
        )
        .addFilterBefore(jwtAuthenticationFilter, UsernamePasswordAuthenticationFilter.class);
    return http.build();
}
⚠️ Security Note: The /api/bookings/my and /api/orders/my endpoints are NOT restricted to ADMIN — any authenticated customer can fetch their own bookings and orders only.

4. Groq LLM Integration — AI Feedback & Culinary Assistant

4.1 GroqService — API Call with Fallback

GroqService.java JAVA 
@Service
@Slf4j
public class GroqService {

    @Value("${app.groq.api-key:NO_KEY}")
    private String apiKey;

    private static final String GROQ_API_URL =
            "https://api.groq.com/openai/v1/chat/completions";
    private static final String GROQ_MODEL = "llama3-8b-8192";

    public String callGroq(String prompt) {
        // Graceful fallback if no API key configured
        if ("NO_KEY".equals(apiKey) || apiKey.trim().isEmpty()) {
            log.warn("Groq key not configured. Using cinematic local backup.");
            return generateFallbackResponse(prompt);
        }
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        headers.setBearerAuth(apiKey);

        Map<String, Object> body = Map.of(
            "model", GROQ_MODEL,
            "messages", List.of(Map.of("role", "user", "content", prompt)),
            "temperature", 0.3
        );
        ResponseEntity<Map> resp = restTemplate.postForEntity(GROQ_API_URL,
                new HttpEntity<>(body, headers), Map.class);

        List choices = (List) resp.getBody().get("choices");
        Map msg = (Map) ((Map) choices.get(0)).get("message");
        return (String) msg.get("content");
    }
}

5. Dashboard Revenue Endpoint

DashboardController.java — getRevenue() JAVA 
@GetMapping("/revenue")
public Map<String, Object> getRevenue() {
    List<Order> orders = orderRepository.findAll();
    Map<YearMonth, Double> revenueMap = new HashMap<>();
    for (Order o : orders) {
        YearMonth ym = YearMonth.from(o.getCreatedAt());
        revenueMap.merge(ym, o.getTotalAmount(), Double::sum);
    }
    Map<String, Double> formatted = revenueMap.entrySet().stream()
        .collect(Collectors.toMap(
            e -> e.getKey().toString(),
            Map.Entry::getValue()));
    return Map.of("monthlyRevenue", formatted);
}
